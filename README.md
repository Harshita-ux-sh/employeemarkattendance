Sentinel: Biometric Attendance & Payroll Dispatch Hub

Sentinel is a high-fidelity, single-page, enterprise-ready biometric employee logging terminal, identity validator, roster management system, and automated payroll gateway. It is designed to modernize workforce management with robust visual facial pattern analysis simulations and clean corporate administration controls.

---

## 🌟 Key Functional Modules

### 1. Attendance Terminal (Kiosk Gateway)
* **Real-time Camera Feed:** Stream local video via high-performance HTML5 canvas mapping back with mirrored alignment configurations.
* **Biometric Mesh Overlays:** Advanced 3D structural marker meshes simulated over live video layout sectors.
* **Anti-Spoofing & Liveness Guard:** Simulated machine-learning validation routines scoring head tilts, expressions, and environment backlighting metrics over standard frame requests.
* **Biometric Database Lookup:** Fully automated "Facial Matrix" matching checks with option for manual administrative overrides.

### 2. Live Monitoring Console (Dashboard Overview)
* **Operational KPI Strip:** High-contrast summary tiles tracing active workforce metrics, biometric check-in counts, late check-in warning rates, and overall SLA compliance scores.
* **Historical Heatmap Charting:** Custom inline vector-line bar charts visually tracking employee check-ins, tardiness, and absences over a rolling 7-day work cycle.
* **Real-time Captures Feed:** Structured audit history detailing recent check-in attempts along with archived frame snapshots, matching scores, and spoof diagnostic notes.

### 3. Corporate Roster (Employee Directory)
* **Advanced Multi-Filter Search:** Instantly screen personnel indexes by registration department, contract classification, or check-in status.
* **Facial Capture Onboarding:** Enroll new employees directly through an automated administrative webcam sub-terminal.
* **Digital Profile Cards:** View clean biometric tags, custom-rendered initial fallbacks, direct email handles, and department codes.

### 4. Financial Calculations (Payroll scheme)
* **Automated Deduction Deductibles:** Directly updates monthly payments reflecting biometric late flags (-$50/occurrence) and absolute absences (-$150/occurrence).
* **OT Incentive Compensation:** Verifies biometric active records to compute overtime bonuses (+$150/occurrence).
* **Direct Disbursement Control:** Single-click modern ACH batch transfer simulation locks records and logs historical transactions with automated system authorization code stamps.
* **Payslip Generator:** Individual interactive payslip window with printing optimizations and AES cryptographic signature details.

### 5. Document Center (Reports Hub)
* **PDF Export Framework:** Instantly downloads structured corporate summaries listing operational activity.
* **Excel Spreadsheet Mapping:** Auto-compiles active biometric records, liveness scores, and payroll ledgers into multi-sheet spreadsheets ready for business imports.

## 🛠️ Built With

* **Framework:** React 18+ (bundled via Vite)
* **Type System:** TypeScript for reliable state architecture
* **Styles:** Tailwind CSS (modern light-theme slate colors, sharp typography)
* **Animations:** Framer Motion (`motion/react`) for smooth dashboard route transitions
* **Icons:** Lucide React

## ⚙️ Getting Started

### Prerequisites
* Node.js (v18.0.0 or higher)
* Active camera input (required for full interactive kiosk features)

### Installation
1. Install project dependencies:
   ```bash
   npm install
   ```

2. Spin up the local development server:
   ```bash
   npm run dev
   ```

3. Open the application in your browser:
   ```
   http://localhost:3000
   ```

### Quality Assurance & Linting
Validate the TypeScript compilation and code styles cleanly with:
```bash
npm run lint
```
